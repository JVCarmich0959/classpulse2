# ClassPulse

## Behavioral Intelligence for Schools

ClassPulse is a lightweight analytics system that turns classroom behavior logs into clear, actionable insights for teachers and administrators.

## What It Does
Standardizes behavior tracking across classrooms
Aggregates incident data in real time
Visualizes trends by time, class, grade, and teacher
Highlights patterns that inform intervention decisions

## Architecture

![ClassPulse Architecture](assets/architecture.png)

### Stack
* Frontend: Vite + Vanilla JS (SPA)
* Backend: Supabase (Auth + Database + REST)
* Serverless: Supabase Edge Functions
* Deployment: Netlify

## Why It Matters

Most school behavior systems produce data without insight.
ClassPulse **makes patterns visible—so decisions can be based on evidence**, not assumptions.
